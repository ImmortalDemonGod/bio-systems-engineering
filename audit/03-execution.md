- `python -c "calculate_decoupling(one_row_df, zones)" with 1-row DataFrame having hr/dist/dt columns`: Raises ZeroDivisionError: 'float division by zero'. The cumsum midpoint split produces an empty first_half DataFrame when work_df has exactly one row with dt=5s (elapsed=[5], midpoint_s=2.5, first_half=work_df[elapsed<=2.5]=empty). first_half['dt'].sum()=0.0 causes float division by zero at ef_1 computation (metrics.py:205-208). No guard exists in calculate_decoupling. Note: calling with a DataFrame missing expected columns raises KeyError instead; the ZeroDivisionError only manifests when all required columns (hr, dist, dt) are present. (`src/biosystems/physics/metrics.py:199-213`)
